# Spritz SDK Examples

Standalone example apps that use the [Spritz SDK](../../README.md). Each example is self-contained and depends on the SDK from the repo root.

| Example | Description |
|--------|-------------|
| [vite-react-app](./vite-react-app) | Vite + React app with Spritz email auth and chat (channels + messages) |

## Running an example

1. From the repo root: `npm install && npm run build`
2. `cd examples/<example-name>`
3. `npm install` then follow the example’s own README (e.g. set `VITE_SPRITZ_API_KEY`, run `npm run dev`)
